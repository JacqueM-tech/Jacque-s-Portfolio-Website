I got started on this journey by reviewing HTML, CSS, and JavaScript tutorials on YouTube to satisfy my burning curiosity for web development. I found it very intriguing and wanted to learn how to build websites.

That curiosity then opened a door that could never be shut. When I learned how to build my first website, I realized I found something that gave me the freedom, versatility, and the feeling of accomplishment that I was looking for in my work. Now I've made a full switch to front-end development, where I can utilize my coding skills along with my project management skills and an eye for details to WOW my clients.
